📖 Logique de composition
Comment sont composées les équipes ?

Score global = moyenne de 💪HC + 🦵MI + 🔥SA + ❤️CA divisé par 4.

▶ Serpentin : le n°1 de chaque équipe a un score équivalent, idem n°2, etc.
▶ Équilibre des dominantes : les tops (4-5★) sur chaque critère sont répartis entre les équipes — pas regroupés ensemble.
▶ Genre équilibré : ratio ♂/♀ harmonisé entre les équipes.
